# Самый простой Telegram-бот

Это первая учебная версия бота:

- команда `/start`;
- стартовая картинка с рыбаком;
- текст `Привет, рыбак!Закидывай удочку и любуйся своим уловом`;
- две inline-кнопки без текста: `🌊` и `🌊`;
- одно меняющееся сообщение: после выбора меняются картинка, текст и кнопки;
- уже выбранные кнопки отмечаются кастомными эмодзи и не превращаются обратно в `🌊`;
- у пользователя есть 5 попыток открыть новые кнопки;
- под описанием рыбы показывается цитата с количеством оставшихся попыток.

## 1. Что лежит в папке

`bot.py` — главный файл. В нём написано, что бот делает.

`requirements.txt` — список библиотек, которые нужны боту.

`.env.example` — пример файла с токеном. Токен — это секретный ключ от BotFather, по которому Telegram понимает, каким ботом вы управляете.

`assets/` — папка с картинками для бота.

`.gitignore` — список файлов, которые не стоит случайно отправлять наружу. Например, настоящий токен лучше не публиковать.

## 2. Как получить токен

1. Откройте Telegram.
2. Найдите `@BotFather`.
3. Напишите ему `/newbot`.
4. Придумайте имя бота.
5. Придумайте username бота. Он должен заканчиваться на `bot`, например `my_fish_slots_bot`.
6. BotFather пришлёт токен. Это длинная строка вроде `123456:ABC...`.

## 3. Как подготовить проект

Откройте терминал в этой папке и выполните команды ниже.

Создать отдельное место для библиотек:

```bash
python3 -m venv .venv
```

Включить это место:

```bash
source .venv/bin/activate
```

Установить библиотеку для Telegram:

```bash
pip install -r requirements.txt
```

## 4. Как добавить токен

Создайте файл `.env` рядом с `bot.py`.

Внутри напишите:

```bash
BOT_TOKEN=ваш_токен_от_BotFather
```

Бот сам прочитает токен из файла `.env` при запуске.

## 5. Как запустить бота

```bash
python bot.py
```

Если всё хорошо, вы увидите:

```text
Бот запущен. Остановить можно клавишами Ctrl+C.
```

После этого откройте своего бота в Telegram и отправьте ему:

```text
/start
```

## 6. Что означает код простыми словами

`START_TEXT` — текст, который бот показывает после команды `/start`.

`PERCH_TEXT` — текст, который бот показывает после первой кнопки `🌊`.

`PIKE_TEXT` — текст, который бот показывает после второй кнопки `🌊`.

`MAX_ATTEMPTS` — сколько раз пользователь может открыть новые кнопки.

`build_caption()` — функция, которая добавляет к описанию рыбы строку с оставшимися попытками.

`build_keyboard()` — функция, которая делает кнопки. Если пользователь уже выбирал кнопку, она показывает на этой кнопке кастомный эмодзи.

`start()` — функция для команды `/start`. Она показывает стартовую картинку, приветствие и две кнопки.

`choose_slot()` — функция для нажатий на кнопки. Она смотрит, какую кнопку нажали, и меняет то же самое сообщение: картинку, текст и кнопки.

`main()` — место, где бот собирается и запускается.

`load_dotenv()` — команда, которая читает файл `.env`, чтобы бот нашёл токен.

`CommandHandler("start", start)` — строка, которая говорит: когда пользователь пишет `/start`, нужно вызвать функцию `start()`.

`CallbackQueryHandler(choose_slot)` — строка, которая говорит: когда пользователь нажимает inline-кнопку, нужно вызвать функцию `choose_slot()`.

`app.run_polling()` — команда, которая постоянно спрашивает Telegram: “Есть ли новые сообщения или нажатия кнопок?”

Остановить работающего бота можно в терминале клавишами `Ctrl+C`.
